# Configuring RPi

## Raspberry Pi 3

1. Flash Raspbian on SD card

 - Create an empty file `ssh` in boot partition

2. [Headless wired LAN] Log in to RPi

 - ip can be taken from the router GUI
 - defaul password: raspberry
 - run `ssh pi@rpi-ip-addr`
